Triggers

  • User hovers on/off product image tile
  • User hovers on/off favorite heart
  • User hovers on/off quick shop button
  • User clicks on quick shop button
  • User hovers on/off size or quantity option buttons
  • User clicks on quantity option buttons
  • User clicks on modal exit button

Rules

User hovers on product image tile

  • Initial product image slides to the left while another image slides in from right to replace it
  • Quick shop bar overlay slides from off screen left to visible right
  • Favorite heart scales horizontally from 0 to 100% centrally
  • Product title quickly fades from a black color to blue gray font color

User hovers off of product image tile

  • Second product image slides to the right while inital image slides from left to take its place
  • Quick shop bar overlay slides from right to left and is no longer visible
  • Favorite heart scales horizontally from 100% to 0 centrally and is no longer visible
  • Product title quickly fades from a blue gray color back to black font color

User hovers on favorite heart

  • User's cursor changes from default arrow to hand
  • Heart icon's gray fill fades to black fill

User hovers off favorite heart

  • User's cursor changes back from hand to default arrow
  • Heart icon's black fill fades to gray fill

User hovers on quick shop button

  • User's cursor changes from default arrow to hand
  • Overlay slides over from left to right to invert background color and text color

User hovers off quick shop button

  • User's cursor changes from hand to default arrow
  • Overlay slides back from right to left and the inverted layer is hidden, revealing the default black text and transparent white background color

User clicks on quick shop button

  • Modal containing product information along with new information (size options, quantity options, exit button, and add to cart button) expands in a horizontal direction from center

User hovers on any size / quantity option button

  • User's cursor changes from default arrow to hand
  • Black background color moves left to right as the black text fades to white

User hovers off any size / quantity option button

  • User's cursor changes from hand to default arrow
  • Black background color moves right to left and is followed by the default white background color as the white text fades back to default black text

User clicks on quantity option buttons

  • The quantity instantly will increase or decrase
  • The number will remain within the maximum 5 and the minimum 1

User clicks on modal exit button

  • The modal instantly disappears
