https://www.urbanoutfitters.com/mens-jackets

Triggers

  • User hovers on or off of product image tile
  • User hovers on or off color option
  • User clicks quick shop button
  • User clicks a color option
  • User clicks on exit quick shop modal
  • User hovers on quick shop modal buttons
  • User hovers on favorite button
  • User clicks on favorite button

Rules

User hovers on or off of product image tile

  • Initial product image instantly changes to another product view (either on model or product view)
  • On hover off, new image instantly changes back to default image that was present before hover
  • Quick shop button and favorite button overlays on top of new product image and on hover off, they instantly disappear

User hovers on or off color option

  • On hover, instant light gray stroke appears around color option
  • Off hover, instant light gray stroke disappears around color option

User clicks quick shop button

  • modal with more information and buttons quickly fades in and scales up to default size on click
  • Before information loads, modal is completely white with simple loading animation (grey small circles that form a circular shape, while one small grey circle becomes darker for a very small amount of time then the one next to it does and same and repeats until information is loaded)

User clicks a color option

  • On click, product image changes to an image of the product variant in that color
  • Light grey stroke changes to black stroke of same stroke weight instantly on click
  • Color selected will stay selected, if user opens quick shop modal

User clicks on exit quick shop modal

  • modal closes instantly
  • If color was changed within modal, then color selection correspondes to product tile once modal is closed

User hovers on quick shop modal buttons

  • modal buttons fade to less saturated color
  • On hover off, modal buttons fade back to default full saturated color

User hovers on favorite/heart button

  • heart icon instantly changes from black stroke to grey stroke
  • heart icon instantly changes from grey stroke back to default black stroke

User clicks on favorite/heart button

  • If unfavorited, on click the heart instantly fills with the stroke color (grey on hover and black off hover) with no apparent stroke color
  • If favorited, on click the heart instantly reverts back to white fill and default stroke color (grey stroke on hover and black stroke off of hover)
