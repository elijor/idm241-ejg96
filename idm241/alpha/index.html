https://www.urbanoutfitters.com/mens-jackets

Triggers

  • User hovers on or off of product image tile
  • User hovers on or off color option
  • User hovers quick shop button
  • User clicks quick shop button
  • User clicks a color option
  • User clicks on exit quick shop modal
  • User hovers on quick shop modal buttons
  • User hovers on favorite button
  • User clicks on favorite button

Rules

User hovers on or off of product image tile

  • Initial product image quickly slides simultaneously while another image takes its place
  • On hover off, the initial image slides back, reverse direction of the hover on transition
  • Quick shop button fades in as an overlay over the product image and fades off as user hovers off

User hovers on or off color option

  • On hover, color option/icon transitions to a slightly larger scale
  • Off hover, color option transitions back to the default scale

User hovers quick shop button

  • Quick shop button fades to a different color and transitions to a slightly larger scale
  • On hover off, color fades back to default color and scales back down to default size

User clicks quick shop button

  • modal with more information and buttons opens on click
  • modal fades in and scales from smaller to larger quickly

User clicks a color option

  • Color icon scales back to default size with a bold border fades into view
  • Product image transitons to product image assigned to that color
  • Color selected will stay selected, if user opens quick shop modal

User clicks on exit quick shop modal

  • modal closes by fading out and scaling down quickly
  • If color was changed within modal, then color selection correspondes to product tile once modal is closed

User hovers on quick shop modal buttons

  • modal buttons fades background color and text color to reverse
  • On hover off, the transition reverses and button fades back to default background color and text color

User hovers on favorite/heart button

  • Icon fades to a different fill (50%) color with a small scale up to a slightly larger size
  • On hover off, the icon fades to default fill color and scales back down to default size

User clicks on favorite/heart button

  • Icon fades back to different fill (100%) color and scales back to default size
  • If the icon is already selected, clicking it again will deselect it, and the icon will scale down and fade back to default / unselected color and scale back to default size
