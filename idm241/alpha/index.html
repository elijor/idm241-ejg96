Urban Outfitters Product Tile

IDM241 Alpha Description

Urban Outfitters Product Tiles Link

Triggers

  • User hovers on and off of product image tile

Rules

User hovers on product image tile

  • Initial product image instantly changes to another product view (either on model or product view)
  • Quick shop bar/button and favorite button instantly overlays on top of new product image

User hovers off of product image tile

  • New image instantly changes back to default image that was present before hover
  • Quick shop bar/button and favorite button instantly disappears from product image

Feedback
